# Mole

Herramienta de limpieza y optimización para macOS.

## Instalación

```bash
brew install tw93/tap/mole
```

## Comandos

| Comando | Descripción |
|---------|-------------|
| `mo` | Menú principal |
| `mo clean` | Liberar espacio en disco |
| `mo uninstall` | Desinstalar apps completamente |
| `mo optimize` | Verificar y mantener el sistema |
| `mo analyze` | Explorar uso de disco |
| `mo status` | Monitorear salud del sistema |
| `mo purge` | Eliminar artefactos de proyectos antiguos |
| `mo touchid` | Configurar Touch ID para sudo |
| `mo update` | Actualizar a la última versión |
